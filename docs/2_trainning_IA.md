## 2. 페이지별 상세 구조

### 2.1 홈페이지 (/)

```markdown
# 홈페이지 구성요소

## Header
- 로고: "신승반점 개발 실습"
- 네비게이션: 학습과정 | 가이드 | 실습 | 커뮤니티 | 내 진도
- CTA 버튼: "지금 시작하기"

## Hero Section
- 메인 카피: "코딩을 한 번도 해보지 않은 사람도 3시간 안에 멋진 웹사이트를 만들 수 있다"
- 서브 카피: "100% 성공 보장형 완전초보자 맞춤 실습 플랫폼"
- 주요 통계: 완료율 95% | 평균 학습시간 3시간 | 만족도 NPS 80+
- CTA 버튼: "무료로 시작하기"
- 미리보기 영상

## Features Section
- SIMPLE 원칙 소개
  - Step-by-Step: 한 번에 하나씩만
  - Immediate Feedback: 즉시 결과 확인
  - Mistake Prevention: 실수 방지
  - Progress Visible: 진행상황 가시화
  - Learn by Doing: 실습 중심 학습
  - Easy Recovery: 쉬운 복구

## Learning Path Section
- 5단계 학습 로드맵 시각화
- 각 단계별 예상 소요시간
- 실시간 진행률 표시

## Success Stories Section
- 학습자 후기
- 완성 작품 쇼케이스
- 전후 비교 (Before/After)

## FAQ Section
- 자주 묻는 질문 10개

## Footer
- 빠른 링크
- 소셜 미디어
- 문의 정보
- 저작권 정보
```

## 5. 사용자 경험 흐름

### 5.1 신규 사용자 온보딩

```markdown
# New User Onboarding Flow

1. **랜딩** → 서비스 소개 및 가치 제안
2. **등록** → 간단한 회원가입 (소셜 로그인 지원)
3. **환경 체크** → 시스템 요구사항 자동 확인
4. **툴 설치** → 원클릭 자동 설치 가이드
5. **첫 프로젝트** → 신승반점 템플릿 로드
6. **가이드 투어** → 인터페이스 주요 기능 소개
7. **실습 시작** → 첫 번째 단계 진입
```

### 5.2 학습 진행 흐름

```markdown
# Learning Progress Flow

각 단계별 공통 흐름:
1. **단계 소개** → 학습 목표 및 예상 시간 안내
2. **사전 체크** → 이전 단계 완료 확인
3. **가이드 확인** → 단계별 상세 설명 제공
4. **실습 진행** → 실시간 코딩 및 미리보기
5. **자동 검증** → 코드 정확성 체크 및 피드백
6. **완료 확인** → 체크포인트 통과 및 다음 단계 준비
7. **성취 인정** → 단계 완료 축하 및 진행률 업데이트
```

### 5.3 에러 복구 흐름

```markdown
# Error Recovery Flow

1. **에러 감지** → 자동 코드 검증 시스템
2. **에러 분석** → 문제 유형 및 원인 파악
3. **해결 제안** → 자동 수정 옵션 제공
4. **선택 처리** → 자동 수정 vs 직접 수정
5. **백업 복원** → 필요시 이전 버전으로 롤백
6. **재검증** → 수정 후 다시 확인
7. **학습 계속** → 정상 플로우로 복귀
```

이 IA 구조는 PRD에서 제시한 **SIMPLE 원칙**과 **100% 성공 보장**이라는 핵심 가치를 반영하여, 완전 초보자도 실패 없이 웹개발을 학습할 수 있도록 설계되었습니다.
